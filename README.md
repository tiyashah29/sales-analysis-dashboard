# Sales Analysis Dashboard (Excel)

This project is an Excel-based dashboard created to analyze sales data and derive meaningful insights.

## Key Features:

* KPI Analysis: Total Sales, Total Profit, Total Orders
* Sales by Category (Bar Chart)
* Sales by Region (Pie Chart)
* Monthly Sales Trend (Line Chart)

## Insights:

* Technology is the top-performing category
* West region contributes the highest sales
* Sales show an increasing trend over time with seasonal patterns

## Tools Used:

* Microsoft Excel
* Pivot Tables
* Data Visualization

## Purpose:

To demonstrate data analysis and visualization skills for real-world business scenarios.


## SQL-Based Data Analysis

In addition to Excel dashboarding, SQL was used to perform deeper data analysis and extract insights directly from the dataset.

### Key SQL Queries & Insights:

**1. Sales by Category**

* Used GROUP BY to calculate total sales per category
* Insight: Technology category generates the highest revenue

**2. Sales by Region**

* Aggregated sales across different regions
* Insight: West region contributes the highest sales

**3. Top 5 Products**

* Identified top-performing products using ORDER BY and LIMIT
* Insight: A small number of products contribute significantly to total revenue

**4. Monthly Sales Trend**

* Analyzed time-based trends using date functions
* Insight: Sales show an overall increasing trend with seasonal fluctuations

### SQL Concepts Used:

* SELECT, WHERE, GROUP BY
* Aggregation functions (SUM, COUNT)
* ORDER BY, LIMIT
* Basic data filtering and analysis

### Tools Used:

* SQLite (Online SQL Editor)
